Local-first developer CLI

DustVault

DustVault is a local-first developer CLI for turning codebases into AI-friendly bundles and inspecting project snapshots offline. It supports project ingest, bundle export, diff, search, extract, and tree-based inspection workflows without requiring a hosted service.

What DustVault does

Package codebases for AI workflows

Turn local projects into portable, AI-friendly bundles that are easier to inspect, review, and reuse.

Inspect snapshots offline

Use diff, search, extract, stats, and tree tools to inspect project state without relying on a hosted backend.

Keep everything local

Run on your own machine with local storage and local workflows, backed by SQLite for vault operations.

Core features

  • Ingest local projects into a SQLite-backed vault
  • Export AI-friendly code bundles from local source trees
  • Inspect bundles with diff, search, extract, tree, and stats commands
  • Work locally without a required hosted service
  • Keep project snapshots portable and easy to review

Example commands

dustvault ingest ./my_project
dustvault export-bundle ./my_project
dustvault search bundle.txt "FastAPI"
dustvault tree bundle.txt
dustvault diff old_bundle.txt new_bundle.txt
